联合体

Aug 09, 202517 min read

联合体（Union）是 C++ 中的一种特殊数据类型，它允许在同一内存位置存储不同类型的数据。联合体的所有成员共享同一块内存空间，因此在任何时刻只能存储其中一个成员的值。
