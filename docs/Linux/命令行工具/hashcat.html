Hashcat 完整详细使用指南

一、基本介绍

Hashcat 是世界上最快的密码恢复工具，支持 CPU 和 GPU 加速。

特点

  • 支持 300+ 种哈希算法
  • 多平台支持（Windows/Linux/macOS）
  • 支持分布式破解
  • 高度优化的内核
  • 支持实时进度保存和恢复
