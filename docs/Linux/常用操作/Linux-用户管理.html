Linux 用户基础及其管理详解

一、用户账户基础概念

1.1 用户类型分类

Linux系统中的用户主要分为三类：

超级用户（root）

  • UID为0
  • 拥有系统最高权限
  • 可以执行任何操作

系统用户

  • UID通常为1-999（不同发行版可能不同）
  • 用于运行系统服务和守护进程
  • 通常不允许登录

普通用户

  • UID从1000开始
  • 权限受限
  • 日常工作使用的账户
